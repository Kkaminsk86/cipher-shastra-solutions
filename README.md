## **Cipher Shastra challenge solutions**

## **Setup**

### 1. Clone or download this repository to your local machine

### 2. Install project dependencies:

```shell
npm install
```

### 3. Create .env file. Place here your Alchemy API_KEY => See '.env_example' file.

### 4. Compile contracts:

```shell
npx hardhat compile
```

### 5. Solutions are in 'scripts' folder. Run each script by the use of command:

```shell
npx hardhat run scripts/*name_of_challenge.js
```
